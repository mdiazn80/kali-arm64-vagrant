## Introduction

## Requirements

 - https://www.vagrantup.com/
 - https://www.vmware.com/es/products/fusion.html

Install vagrant-vmware-desktop plugin
```
$ vagrant plugin install vagrant-vmware-desktop
```

## Init

Start
```
$ vagrant up
```

Stop 

```
$ vagrant halt
```

Login 

```
$ vagrant ssh
```
